Syntax

A grammar is specified via a formalism similar to a Backus-Naur Form grammar, though the point of grammars is not to parse BNF grammars or express context-free grammars but to create useful character sequence matchers. Consequenty, no claims are made as to the purity of grammars as CFGs or whatever. Basically, a grammar is a collection of matching rules, where every rule specifies a pattern of terminal expressions -- character sequences -- or non-terminal rules.
