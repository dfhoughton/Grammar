Download

A source tarball with documentation, ant build script, etc: grammar.tgz (179K)
For the most up-to-date code, clone the github repository: 
	git clone https://github.com/dfhoughton/Grammar.git
Once you have the code, most things you may want to do can be accomplished with the build file. 
houghton@NorthernSpy:/tmp/Grammar$ ant -p
Buildfile: /tmp/Grammar/build.xml
Recursive, compositional pattern matching for Java.
Main targets:

 clean    removes anything created by any other ant task
 compile  compile source to bytecode
 jar      packages code into jar
 javadoc  API for public and protected methods
 tar      bundle up the source and documentation
Default target: jar
If you wish to run the unit tests, you will need JUnit 4 on your classpath.
